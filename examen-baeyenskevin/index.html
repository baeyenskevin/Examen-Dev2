Oefening 1

Toon een lijst van 10 smartphones. Zorg ervoor dat je de lijst herhaaldelijk kan filteren volgens het merk en de minimum + maximum prijs.

Starter code

Je vindt de starters code terug in de oefening-1-objects-and-arrays folder. Schrijf je JavaScript in oefening-1-objects-and-arrays/index.js.

Demo uiteindelijke resultaat

Criteria

  • Alle smartphones worden getoond.
    2 punten
  • De smartphones kunnen herhaaldelijk gefiltered worden volgens het merk.
    3 punten
  • De smartphones kunnen herhaaldelijk gefiltered worden volgens de minimum en maximum prijs.
    3 punten

Code qualiteit

  • De code is volledig objectgeoriënteerd geschreven
    4 punten
  • Er wordt geen code herhaalt.
    2 punten

Totaal: 14 punten

Oefening 2

Maak een webapplicatie waar je de landen die je al bezocht hebt kan bijhouden. Een lijst van landen kan je ophalen via de volgende API request URL: https://restcountries.eu/rest/v2/all. Zorg ervoor dat de landen gegroepeerd zijn volgens het continent (regio) dat ze toe behoren, landen binnen hetzelfde continent worden alfabetisch gesorteerd.
Een gebruiker kan aanduiden via een checkbox welke landen ze al bezocht hebben, een aangevinkte checkbox kan ook terug uitgevinkt worden. Er is ook een counter beschikbaar die aantoont hoeveel landen er in totaal al bezocht zijn. (je hoeft de aangeduide landen niet permanent op te slaan)

Starter code

Je vindt de starters code terug in de oefening-2-async folder. Schrijf je JavaScript in /oefening-2-async/index.js.

Demo uiteindelijke resultaat

Criteria

  • Via een API call wordt er landen data binnengehaald en de landen worden getoont met de naam en de juiste vlag
    3 punten
  • De opgelijsten landen zijn gegroepeerd volgens continent en binnen hetzelfde continent zijn de landen alfabetisch geranschikt
    5 punten
  • Wanneer de gebruiker een land aanvinkt, gaat de counter naar omhoog, wanneer de gebruiker een land afvinkt gaat de naar omlaag
    5 punten

Code qualiteit

  • De code is volledig objectgeoriënteerd geschreven
    7 punten
  • Er wordt geen code herhaalt.
    3 punten

Totaal: 23 punten
